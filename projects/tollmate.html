Tollmate

Download APK

The android app is a client side app which is used to send request to the database for automation of toll plaza payment process. Parallelly when the user drags and drops the icon, the apps is made to run in the background and the continuous data transmission of the present location (latitude and longitude) is send to the database to process it.
