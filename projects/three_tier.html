Modern three tier ATM Authentication

Full Paper (PDF)

A conceptual Idea to improve the security in ATM authentication.
The proposed security system consists of three tiers or levels of credential verification.

Each of these levels has their own mechanism

1. Holographic Keyboard
It will consist of the randomised keyboard layout in holographic which can be viewed only from a certain angle. Through this, we can restrict the use of recording the pin typed physically.

2. QR Code scanning
Once the pin is authenticated, a QR is generated on the screen and it will be scanned by the user using their banking application.

3. Fingerprint-based hand scanning
Once the QR is matched, the user must show their hand into a holographic screen to read the prints.

Once authenticated, the ATM user requirement is performed.
